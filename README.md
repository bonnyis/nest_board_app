# 프로젝트 개요

이 프로젝트는 NestJS를 기반으로 한 백엔드 애플리케이션입니다. Nest 공식 CLI(@nestjs/cli)를 사용해 초기 세팅을 진행했으며, Node.js v20 환경에서 개발되었습니다.

본 프로젝트는 인프런 강의 **「NestJS로 배우는 백엔드 핵심 패턴」**을 학습하며 따라 만든 실습 중심 프로젝트입니다.

강의 링크: https://inf.run/iDbkU

# 기술 스택 (Project Spec)
  Runtime
  - Node.js: v20.x

  Framework
  - NestJS
  - Nest CLI (@nestjs/cli)
  Language
  - TypeScript
  Package Manager
  - npm

# 학습 목적

  1. NestJS 아키텍처 이해 (Module / Controller / Provider)

  2. 의존성 주입(DI) 개념 숙지

  3. 실무에서 사용되는 NestJS 프로젝트 구조 학습

  4. TypeScript 기반 백엔드 개발 흐름 익히기
